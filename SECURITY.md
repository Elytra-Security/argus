# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2026.01 | Yes       |

## Reporting a Vulnerability

Do not open a public issue for security vulnerabilities.

Email: security@elytrasecurity.com

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested remediation

We will acknowledge receipt within 48 hours and provide a remediation
timeline within 5 business days.

## Verifying Release Integrity

Every release tarball is accompanied by a SHA-256 checksum file.
Always verify before installing:

    sha256sum --check argus-VERSION.tar.gz.sha256

## Responsible Disclosure

We follow coordinated disclosure. Please give us reasonable time to
remediate before public disclosure.
